登录方式怎么选？

官方支持两种方式：ChatGPT 登录或 API Key 登录。

方式 适合谁 注意事项
ChatGPT 登录 已经有 ChatGPT Plus、Pro、Business、Edu、Enterprise 等账号的人。 Codex 会打开浏览器让你完成登录，过程中可能需要手机号/电话验证。建议使用本人可长期接收验证码的手机号，不建议使用临时、共享或第三方接码号码。Codex Cloud 需要 ChatGPT 登录。
API Key 登录 想按 API 用量计费，或做自动化、命令行流程的人。 需要到 OpenAI Platform 创建 API Key，费用按 API 账户计费。

桌面版登录

打开 Codex 后，按界面提示选择登录方式。小白优先选 ChatGPT 登录。若出现电话或短信验证，请使用本人可长期接收验证码的手机号；临时虚拟号或共享接码号码可能导致验证失败、账号找回困难或账号安全风险。

开始1

打开一个文件夹

先在电脑里准备一个专门给 Codex 使用的项目文件夹。项目就是一组文件，比如网页、脚本、文档、代码。

  1. 在任意一个硬盘里新建文件夹，例如 D:\codex-projects\my-first-project。名字可以用英文、数字或拼音，尽量不要太长。
  2. 把你想让 Codex 处理的文件放进这个文件夹。刚开始也可以先放一个简单的 txt 文档或网页文件练习。
  3. 打开 Codex 桌面版，把这个文件夹拖到左侧的“项目”区域；如果不能拖入，就点击“添加项目”或“打开文件夹”。
  4. 文件夹出现在左侧项目列表后，点击项目名称即可打开。打开后再开始输入任务。

常用功能介绍

先认识这些按钮和模式，用 Codex 会稳很多。

安全权限

Codex 的三种常见操作权限

权限决定 Codex 能不能读文件、改文件、运行命令。新手建议先用默认权限，不要一开始就给完全访问。

  • 只读：适合聊天、解释、做计划，不让 Codex 改文件。
  • 默认权限：可在当前项目里读写，需要越界时会询问你。
  • 完全访问：权限最大，风险也最高，只给可信项目和明确任务使用。
先计划

计划模式怎么用？

计划模式会让 Codex 先理解项目、问清需求、写出执行方案，再开始动手。适合复杂任务，例如重做页面、修复多个问题、整理项目结构。

  • 在输入框里输入 /plan 可开启计划模式。
  • 也可以先说“先给我计划，不要改文件”。
  • 确认计划后，再让 Codex 开始执行。
整理对话

如何归档？

归档是把旧对话从常用列表里收起来，不等于删除。以后需要时仍可从归档或设置里找回。

  • 桌面版可在对话列表或设置里的归档区域管理。
  • CLI 里可输入 /archive 归档当前会话。
  • 不要把归档和删除混淆，删除会更难恢复。
自动执行

自动化是什么？

自动化可以让 Codex 按时间重复检查任务，例如定时查看项目、提醒进度、跟踪某个长期问题。

  • 在侧边栏的“已安排/自动化”区域查看运行结果。
  • 本地项目自动化需要电脑开机、Codex 正在运行、项目文件还在。
  • 第一次设置前，先在普通对话里测试提示词。
快捷命令

斜杠命令

在输入框里输入 /，会出现可用命令。它像快捷菜单，可以快速切换模式或查看状态。

  • /status：查看当前会话状态。
  • /review：让 Codex 做代码评审。
  • /permissions：调整本次会话权限。
检查结果

查看改动和评审

Codex 改完文件后，不要直接相信结果。先看它改了哪些文件，再决定是否保留。

  • 使用桌面版的改动/评审面板查看差异。
  • 命令行里可用 /diff 查看 Git 改动。
  • 重要项目建议再让 Codex 做一次 /review。

常见问题

遇到问题先看这里，很多情况不是你操作错了。

1. 打不开 OpenAI、ChatGPT 或登录页怎么办？

先确认你的所在地是否在 OpenAI 官方支持范围内，并检查网络是否能正常访问相关服务。本页不提供绕过地区或服务限制的方法。

2. PowerShell 提示脚本被禁止运行怎么办？

Windows 可能限制脚本执行。官方文档提到常见处理方式是把执行策略调整为 RemoteSigned。修改系统策略前，请先阅读微软官方说明。

PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
查看微软执行策略说明

3. Git 功能不可用怎么办？

先安装 Git。Codex 的评审和改动查看功能会用到 Git。Windows 用户也可以用官方文档里的命令安装。

PowerShell
winget install --id Git.Git

4. 输入 codex 后提示找不到命令怎么办？

通常说明 CLI 没安装成功，或安装后终端还没刷新。先关闭并重新打开 PowerShell，再运行 codex --version。如果仍然不行，重新执行 CLI 安装步骤。

5. API Key 应该发给别人吗？

不要。API Key 相当于你的付费钥匙。不要发到聊天、论坛、工单截图或公开仓库里。需要时只在 OpenAI 官方平台创建和管理。
